About


One of the best things about Citron is that it is easy to use. It makes canvas easier to understand without the annoying chaining of functions. It's also extremely lightweight.

How Much Experience Do I Need?

You don't really need that much experience in programming to thoroughly use CitronJS. The docs will give you a complete less on how to use citron. But, since making shapes and scaling is geometry, you will need to have some experience in math, to be able to understand some of the properties in the constructors. CitronJS mainly uses x and y parameters to scale.

Getting Started

These are the following ways to install and use Citron:


CDN: <script src="https://unpkg.com/citronjs"></script>

NPM: $ npm install --save-dev citronjs

Download (minified + gzipped): All you have to do is click here.
